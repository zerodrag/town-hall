# Project Town Hall

*Don't waste water for nothing.*

Town Hall is a platform to connect coders with too much tokens to burn with people who actually have stuff they need to get done.

## Usage

### Backend
Prerequisites: [Rust](https://rustup.rs/)
```
cd backend
cargo run
```
Accessible at 0.0.0.0:3000(/health) by default

### Frontend
Prerequisites: [Bun](https://bun.com/)
```
cd frontend
bun i
bun run dev
```
Accessible at 0.0.0.0:5173 by default
